# 🎉 GatherGo - Event Management System

## 📖 Project Overview

GatherGo is a Full Stack Event Management System developed as part of my BCA Internship.

The application allows administrators to manage events, venues, and users, while users can browse, register, and manage their event registrations.

---

## 🚀 Technologies Used

- React.js
- Flask (Python)
- MySQL
- XAMPP
- HTML
- CSS
- JavaScript

---

## ✨ Features

### Admin

- Manage Users
- Manage Events
- Manage Venues
- View Registrations

### User

- Login
- Browse Events
- Register for Events
- Cancel Registrations
- View Registered Events
